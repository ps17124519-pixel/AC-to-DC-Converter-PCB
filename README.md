# AC to DC Converter PCB Design
## Overview
This project demonstrates the complete design of an AC to DC power supply PCB using KiCad.

The project includes the complete design workflow from schematic capture to PCB layout and 3D visualization.
---
## Features
- Schematic Design
- PCB Layout
- Footprint Assignment
- Interactive Routing
- 3D PCB Visualization
- ERC Verification
- DRC Verification

## Software Used

- KiCad 9

## Design Flow

1. Circuit Design
2. Symbol Placement
3. Footprint Assignment
4. PCB Layout
5. Component Placement
6. Interactive Routing
7. ERC Check
8. DRC Check
9. 3D Model Verification
